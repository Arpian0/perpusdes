Personalization

You can personalize images, home sections, texts from “Admin/Settings/Personalization”. There are several tabs with different controls that help personalize the platform. Let's review each tab functionality.
Page Backgrounds:
There are beautiful backgrounds for different pages that make pages look more beautiful. You can change page backgrounds in this tab.
Home Sections:
To enable/disable homepage sections, you can turn the related button on or off so the section will be displayed or hide.
Hero Styles:
There are two hero styles for Rocket LMS. Hero is the first part of the screen that the user will see when lands on the website. It is very important for this section to be attractive to encourage users to stay on the website.
You can specify a background, title, and description for the full-width hero.
Another hero style is “Animated Hero”. This hero style includes a different layout and design and it is an important extra feature for this part that you can use animated Lottie files and bring awesome and smooth animations to your website.
You can get free Lottie files in .json format from lottiefiles.com and use them on your website.
If you don’t want to use Lottie animations for this hero style, you should disable the related toggle button.
Important: You can specify the hero style from “Admin/Settings/General”. In this part, you can specify which hero style will be loaded.
Video/Image Section:
There is a section on the homepage that includes a background with a “Play” button animation. You can use your favorite assets for this part.
User Panel Sidebar:
In this tab, you can specify the user panel sidebar image and URL.
